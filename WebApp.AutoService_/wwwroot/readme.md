# AutoService Pro - Система управления автосервисом

Веб-приложение для управления автосервисом с модульной структурой и адаптивным дизайном.

## Функциональность

- 📊 Дашборд с ключевыми метриками
- 🚗 Управление справочниками (автомобили, клиенты, работы, запчасти)
- 📋 Документы (заказ-наряды, накладные)
- 🖨️ Печать документов с предпросмотром
- 📱 Полная адаптивность для мобильных устройств

## Установка

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в браузере
3. Приложение работает полностью на стороне клиента
