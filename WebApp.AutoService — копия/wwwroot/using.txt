
## Использование

1. **Начало работы**: При первом запуске создаются демо-данные
2. **Навигация**: Используйте боковое меню для перехода между разделами
3. **Добавление данных**: Кнопки "Добавить" в каждом разделе
4. **Печать**: Кнопка "Печать" в документах открывает предпросмотр
5. **Поиск**: Используйте Ctrl+F для быстрого поиска

## Особенности

- **Модульная архитектура**: Каждый компонент в отдельном файле
- **Адаптивный дизайн**: Оптимизирован для всех устройств
- **Офлайн-работа**: Все данные хранятся в браузере
- **Печать документов**: Профессиональные шаблоны документов
- **Уведомления**: Всплывающие сообщения о действиях

## Технологии

- Vanilla JavaScript (без фреймворков)
- CSS Grid и Flexbox
- LocalStorage для хранения данных
- Chart.js для графиков
- Font Awesome для иконок
